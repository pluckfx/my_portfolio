Kinopoisk Gallery

React-приложение для просмотра топовых фильмов с API Кинопоиска. Реализована загрузка данных с нескольких страниц API, поиск по названию, клиентская фильтрация и пагинация. Во время загрузки отображаются skeleton-карточки. Проект демонстрирует работу с асинхронными запросами (fetch, Promise.all), React Hooks (useState, useEffect) и динамическим рендерингом списков.
